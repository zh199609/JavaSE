interface 接口名称{
	全局变量
	抽象方法
}


接口可以继承多个接口：extends xxx,xxx
抽象类实现接口可以不实现接口的方法
接口不能被实例化
接口不能有构造方法



接口特性
	接口中方的方法都会隐式指定为public abstract
	接口中的变量都会隐式的指定为 public static final
	
	
抽象类和接口的区别
	抽象类的方法可以有方法体，但接口不能
	抽象类中的成员可以是各种类型的，接口只能是public 	static final
	接口中不能含有静态代码块以及静态方法  抽象类可以有
	一个类只能继承一个抽象类，但是可以实现多分接口
